# 🧹 Lead Enrichment & Cleanup Agent

Stop letting messy data ruin your CRM routing and sales outreach. 

Sales teams waste hours manually fixing capitalization, deciphering weird job titles, and hunting down company websites. This autonomous agent takes your raw, unstructured lead data, actively **searches the web** to fill in missing firmographics, normalizes job titles, and flags risky emails—all in seconds.

### ✨ Enterprise Features
* **Live Web Enrichment:** Actively browses the web to find company websites and industry sectors.
* **Taxonomy Normalization:** Converts chaotic titles ("VP of Mktg", "Vice President Marketing") into clean, standard formats ("VP Marketing").
* **Risk Flagging:** Automatically catches fake emails and incomplete profiles before they hit your sales reps' queues.
* **Structured Output:** Delivers ready-to-import Markdown or JSON tables.

### 🚀 Installation (A2A Protocol)
Install globally via skills.sh to enable your local AI agents to use this workflow:

```bash
npx skillsadd mateusoliveiradev1/lead-enrichment-cleanup
```
